# WeatherClientApp
push
Weather Client App project is the UI
This is an Angularjs project.The project uses OpenWeatherMap API to get data. It consists of three components a 'start' , 'weather' and 'Forecast'

The Start component where the user can enter a city name
The input text accepts only Characters and Spaces. If the user enters 123 or special characters the “weather ” and the “Forecast ” will be disabled.

The Weather  component shows the Current weather status
The Forecast Component shows the weather for the up coming 5 days
